Funcões-Métodos

Quando tivermos uma função dentro de um objeto, a função se chamará método.

forma padrão de um objeto contendo um método:

var aluno = {nome: "igor", nota: [5, 8],
media: function (n1, n2){
return (n1 + n2)/2 }
}
console.log (aluno["nome"])
console.log (aluno.media(aluno.nota[0], aluno.nota[1]))

substituindo um método por uma propriedade (variável):

var calcmedia = function (n1, n2){
return (n1 + n2)/2 }
var aluno = {nome: "igor", nota: [5, 8], media:calcmedia }

obs: Isso serve para caso eu queira mudar a forma de como é calculada a média.

this

Eu posso colocar uma função sem argumentos e indicar quais são os argumentos usando o this., isso serve para simplicar na hora de "chamar a função".

Ex:
var calcmedia = function (){
return (this.nota[0] + this.nota[1])/2 }
var aluno = {nome: "igor", nota: [5, 8], media:calcmedia }
console.log (aluno["nome"]) console.log (aluno.media())
